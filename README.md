This project was completed as part of the JP Morgan Chase & Co. Software Engineering Virtual Experience Program on Forage.

## Technologies Used

* Java 17
* Spring Boot
* Apache Kafka
* H2 Database
* REST APIs
* Maven

* Database Integration using JPA
* Incentive API Integration
* User Balance REST API
* End-to-End Testing
